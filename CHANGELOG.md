## [0.1.0] - 2025-05-01

### Added

- ReAct agent loop with tool-calling capabilities
- Plan-Execute agent mode
- Interactive CLI with rich terminal UI
- File operations tools (read, write, search, edit, glob, grep)
- Shell execution with background task support
- Web search and fetch tools
- Code navigator with tree-sitter support
- Memory management with compression and persistence
- Multi-provider LLM support via LiteLLM
- Session resume support
- Model configuration via `~/.aloop/models.yaml`
